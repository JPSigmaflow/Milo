## 📊 Ads-Strategie: $5/Tag Micro-Budget

### Kampagnen-Übersicht

| Parameter | Wert |
|-----------|------|
| **Budget** | $5/Tag ($35/Woche, ~$150/Monat) |
| **Ziel** | Profile Visits → Follows |
| **Format** | Bestes Reel promoten (Boost) |
| **Laufzeit** | Fortlaufend, wöchentlich optimieren |

### Zielgruppe

| Parameter | Einstellung |
|-----------|-------------|
| **Alter** | 18-35 |
| **Sprachen** | Englisch + Spanisch |
| **Interessen** | Pomeranian, Dog lover, Cute animals, Puppies, Pet owner |
| **Erweiterte Interessen** | Dog breeds, Pomeranian (dog breed), Fluffy dogs |
| **Lookalike** | Nach 100+ Followern: Lookalike Audience erstellen |
| **Regionen** | USA, UK, Kanada, Mexiko, Spanien, Lateinamerika |

### Welches Reel promoten?

**Auswahlkriterien (in dieser Reihenfolge):**
1. Höchste organische Engagement-Rate (Likes+Comments/Views)
2. Mindestens 1.000 organische Views
3. Kurz (15-30 Sekunden ideal)
4. Hook in den ersten 2 Sekunden
5. Milo & Lanna zusammen = besser als einzeln

**Empfehlung:** Jede Woche das best-performende Reel der Woche boosten.

### Kampagnen-Setup (Step by Step)

1. **Öffne Instagram App → Profil → Reel auswählen**
2. **"Boost Post" / "Beitrag bewerben" klicken**
3. **Ziel wählen:** "More Profile Visits" / "Mehr Profilbesuche"
4. **Zielgruppe:**
   - "Create Your Own" / "Eigene erstellen"
   - Interessen: Pomeranian, Dogs, Cute animals
   - Alter: 18-35
   - Standort: USA, UK, Canada, Mexico, Spain
5. **Budget:** $5/Tag
6. **Dauer:** 7 Tage (dann evaluieren)
7. **Veröffentlichen**

### Optimierung (wöchentlich)

- **Montag:** Performance der letzten Woche prüfen
- **Kosten pro Profilbesuch** → Ziel: unter $0.10
- **Kosten pro neuem Follower** → Ziel: unter $0.50
- **Wenn CPA zu hoch:** Zielgruppe anpassen oder anderes Reel testen
- **Wenn CPA gut:** Budget auf $7-10/Tag erhöhen
