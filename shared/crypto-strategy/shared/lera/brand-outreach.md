## 🏢 Крупные бренды (5)

### 1. ColourPop Cosmetics
- **Instagram:** [@colourpopcosmetics](https://instagram.com/colourpopcosmetics)
- **Почему подходит:** Affordable, trendy, огромная программа работы с микро-инфлюенсерами. Часто репостят контент от небольших криейторов.
- **Как связаться:** PR-форма на сайте → [colourpop.com/pages/pr](https://colourpop.com/pages/pr) + DM в Instagram с portfolio

### 2. e.l.f. Cosmetics
- **Instagram:** [@elfcosmetics](https://instagram.com/elfcosmetics)
- **Почему подходит:** Активнейшая работа с TikTok/Reels криейторами. Программа "e.l.f. Beautyscape" и постоянный поиск UGC контента. Budget-friendly = идеально для аудитории 18-35.
- **Как связаться:** Influencer portal → [elfcosmetics.com/beautyinfluencer](https://www.elfcosmetics.com) + email: pr@elfcosmetics.com

### 3. NYX Professional Makeup
- **Instagram:** [@nyxcosmetics](https://instagram.com/nyxcosmetics)
- **Почему подходит:** Профессиональный бренд по доступной цене. Программа NYX Professional Makeup Creator Program открыта для микро-инфлюенсеров.
- **Как связаться:** Creator application на сайте NYX + DM + email: NYXInfluencers@loreal.com

### 4. Morphe
- **Instagram:** [@morpheofficial](https://instagram.com/morpheofficial)
- **Почему подходит:** Бренд построен на инфлюенсер-маркетинге. Affiliate program с хорошими условиями даже для небольших аккаунтов.
- **Как связаться:** Affiliate program на сайте morphe.com + DM

### 5. Milani Cosmetics
- **Instagram:** [@milaboratorio](https://instagram.com/milanicosmetics)
- **Почему подходит:** Drugstore бренд с активной PR-программой. Часто работают с микро-инфлюенсерами для honest reviews. Отличные продукты для transformation контента.
- **Как связаться:** Email: influencer@milanicosmetics.com + DM в Instagram
