# 📸 Gaby's Arbeits-System

Organisiert von Pomi 🍟 für Gaby Bazan (Community-Mitarbeiterin @pomesteam)

## 📋 Übersicht

Dieser Ordner trackt **ALLE** Aufgaben, Zusagen und Aktivitäten von Gaby:

### 📁 Dateien

**auftraege.json**
- Alle Aufgaben die Pomi Gaby gibt
- Status tracking (pending/in-progress/done)
- Deadlines & Prioritäten

**zusagen.json**
- Was Gaby zugesagt hat ("Mache heute Fotos", etc.)
- Auto-Tracking ob erfüllt
- Reminder-System 24h vorher

**fotos-tracker.json**
- Alle Fotos die Gaby geschickt hat
- Verwendet für Posts?
- Statistiken (diese Woche, gesamt)

**engagement-log.json**
- Community-Aktivitäten
- Kommentare beantwortet, Posts geteilt
- Engagement-Score

**motivation.json**
- 7 spanische Motivations-Nachrichten
- Rotation-System (Mo/Mi/Fr)
- Response-Tracking

### 🔄 Workflow

1. **Pomi gibt Aufgabe** → `auftraege.json`
2. **Gaby sagt zu** → `zusagen.json`
3. **Gaby liefert** → Update Status + Log in entsprechendem Tracker
4. **Auto-Reminder** wenn Deadline naht

### 📊 Reports

Pomi kann jederzeit Reports generieren:
- Wie viele Aufgaben sind offen?
- Wie ist Gaby's Erfüllungsrate?
- Wie viele Fotos diese Woche?
