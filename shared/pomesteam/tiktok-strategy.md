## Content-Strategie

### Was funktioniert auf TikTok (anders als Instagram):
| TikTok | Instagram |
|--------|-----------|
| Roher, authentischer Content | Polierter, ästhetischer Content |
| Trending Sounds = WICHTIG | Sounds weniger wichtig |
| Text-Overlays beliebt | Captions wichtiger |
| Schnelle Schnitte | Langsameres Pacing OK |
| 15-30 Sek optimal | Bis 90 Sek OK |

### Content-Säulen:
1. **POV Videos** - "POV: Du hast zwei Pomeranians"
2. **Day in the Life** - Tagesablauf der Poms
3. **Travel Content** - Shanghai erkunden
4. **Funny Moments** - Lustige Reaktionen
5. **Trending Sounds** - Aktuelle Trends mit Pom-Twist

## Cross-Posting Workflow

```
1. Video erstellen (ohne Wasserzeichen)
2. → TikTok hochladen (mit TikTok Sounds)
3. → Instagram Reel hochladen (original Audio)
4. → YouTube Shorts (optional)
```

**WICHTIG:** Kein TikTok-Wasserzeichen auf Instagram!
(Instagram straft das ab)
