## ⚠️ Methodology Notes

- **Data source:** Twitter Syndication API (`syndication.twitter.com/srv/timeline-profile/screen-name/{handle}`)
- **Success rate:** ~25 out of ~90 accounts returned data; most Tier 2 accounts returned empty/blocked responses (20 bytes)
- **Noise:** HTML contains hashed/obfuscated tweet IDs that appear as fake handles — these were filtered
- **Limitations:** Syndication API only shows recent tweets; connection strength may be underrepresented
- **Self-mentions filtered:** Accounts mentioning their own brand accounts (e.g., CathieDWood→arkinvest) are noted but represent brand promotion, not peer connections
